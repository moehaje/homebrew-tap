# moehaje Homebrew Tap

Personal Homebrew tap for CLI tools maintained by Moe Haje.

## Install from this tap

```bash
brew tap moehaje/tap
brew install <formula>
```

One-command form:

```bash
brew install moehaje/tap/<formula>
```
